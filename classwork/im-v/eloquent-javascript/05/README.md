# Chapter 05 - Higher-Order Functions

## Setup

Before running this project, create a `keys.json` file in the root of this directory with the following structure:

```json
{
    "mapbox": "YOUR_MAPBOX_API_KEY"
}
```

Replace `YOUR_MAPBOX_API_KEY` with your own Mapbox access token. You can get one by signing up at [mapbox.com](https://www.mapbox.com/).

> **Note:** The `keys.json` file is not included in the repository for security reasons. Each learner must create their own.
